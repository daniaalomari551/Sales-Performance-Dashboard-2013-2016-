# 📊 Sales Performance Dashboard (2013–2016)

This interactive Power BI dashboard analyzes sales data between 2013 and 2016.

## 🧩 Features

- Total Sales (Excluding Tax)
- Total Orders
- Total Quantity Sold
- Total Customers
- Year-over-Year Sales Trends
- Dynamic Filtering by:
  - Year
  - City
  - Customer
  - Product
  - Employee

## 📁 Data Model

The model includes:
- Dim_Date
- Dim_Customer
- Dim_City
- Dim_Employee
- Dim_StockItem
- Fact_Sale
